
# Medikarya Technologies Internship Website

A modern and responsive internship website developed for **Medikarya Technologies**. The website provides an engaging platform for students to explore internship opportunities, learn about the company, and apply through a clean and user-friendly interface.

## 🚀 Features

- Responsive design for all devices
- Modern and clean UI/UX
- Internship program showcase
- Company information section
- Benefits and highlights
- FAQ section
- Contact section
- Smooth scrolling and interactive elements

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 📱 Responsive Design

The website is fully optimized for:
- Desktop
- Tablet
- Mobile

## 📂 Project Structure

```
├── index.html

│   ├── images/
│   └── icons/

```

## 🎯 Purpose

The objective of this project is to provide a professional internship portal where students can discover internship opportunities while experiencing a modern, responsive, and user-friendly interface.
